4. Metrics calculated:

   - Pearson Correlation: Linear relationship strength
   - Spearman Correlation: Rank-order agreement

SUMMARY & RECOMMENDATIONS
================================================================================

STRENGTHS:

PI (AssemblyAI + DeepSeek):
  ✓ Semantic content analysis via LLM
  ✓ Comprehensive feedback generation
  ✓ CEFR proficiency level classification
  ✓ Handles off-topic/incomplete responses
  ✓ Better for general English proficiency assessment

SpeechOcean (Whisper + ML):
  ✓ Fully offline-capable (no API dependency)
  ✓ Lower latency and cost
  ✓ Objective acoustic-based metrics
  ✓ Trained on expert-labeled data
  ✓ Better for pure pronunciation scoring

RECOMMENDATION:
--------------------------------------------------------------------------------
🏆 PI WINS (AssemblyAI + DeepSeek approach)

Use PI if you prioritize:
  • Comprehensive proficiency assessment
  • Detailed written feedback
  • CEFR certification standards
  • Semantic content understanding

HYBRID APPROACH:
--------------------------------------------------------------------------------
Consider combining both:
  • Use SpeechOcean for fast, objective pronunciation metrics
  • Use PI for comprehensive proficiency assessment & feedback
  • Blend scores for robust evaluation
